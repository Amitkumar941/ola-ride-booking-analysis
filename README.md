# Ola Ride Booking Analytics Dashboard

## Project Objective

Analyze ride booking data to understand booking demand, revenue performance, vehicle usage, and cancellation behavior.

## Tools Used

- Excel
- SQL (MySQL)
- Power BI

## Process

1. Cleaned raw data using Excel.
2. Performed analysis using SQL.
3. Built interactive dashboards in Power BI.

## Dashboard Pages

### Overall Analysis
- Total Bookings
- Total Booking Value
- Booking Status Distribution
- Ride Volume Over Time

### Vehicle Type Analysis
- Booking Value by Vehicle
- Average Distance
- Successful Booking Value

### Revenue Analysis
- Revenue by Payment Method
- Top Customers

### Cancellation Analysis
- Driver Cancellation Reasons
- Customer Cancellation Reasons

## Key Insights

- Most rides were completed successfully.
- Some vehicle categories generated higher revenue.
- UPI and Cash were the most used payment methods.
- Cancellations mainly occurred due to driver and customer-related issues.
